# 常见问题

博客项目搭建和使用过程中需要的问题及其解决办法

# 问题

# vuepress

问题: Duplicate keys detected: '/'. This may cause an update error.

原因: 路由重复导致

参考: 1 (opens new window)


问题: 自定义组件以及引入三方组件问题。Error: render function or template not defined in component; vuepress 编译不通过 ReferenceError: self is not defined; ReferenceError: window is not defined;

  • 依然存在问题，页面初次进入组件正常展示。页面刷新后组件消失，切换到其他页面再切换回来会正常显示。

原因: VuePress 是在Node.js 服务端渲染，node没有window，所以报错ReferenceError: document is not defined

解决: 将组件包裹在内置的组件 <ClientOnly> 中; 直接使用自定义组件名称即文件名称; 注意刷新页面后数据丢失, 需要解决重新渲染问题。

参考: 1 (opens new window) 2 (opens new window) 3 (opens new window) 4 (opens new window)


问题: [Vue warn]: Avoid using non-primitive value as key, use string/number value instead.

原因: 避免使用非基本值作为键，而是使用字符串/数字值，报错的意思是 不要用对象或是数组作为key，用string或value作为key

解决: v-for 中 key 使用数组唯一序列，例如索引

参考: 1 (opens new window)
