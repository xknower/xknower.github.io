    # 博客项目搭建

    # 一、思路架构

    第一步, 选型并搭建博客

    1. 学习使用 vuepress 静态网站生成器功能搭建博客
    2. 基于vuepress搭建静态博客, 构建简单博客内容, 发布到 github.io, 并使用自己域名

    第二步, 内容架构及功能优化

    1. 思考博客内容架构, 输出内容架构图, 并修改博客提交相关内容
    2. 思考博客布局, 并优化博客布局及其显示
    3. 整理博客静态网站功能, 预留相关功能: 多语言国际化、搜索栏等

    第三步, 整理提交博客内容

    1. 根据内容架构, 提交博客内容并长期维护

    # 2. 博客发布

    使用Github Pages (opens new window)发布静态网站

    1. github 上新增仓库: xknower.github.io 和 xknower-blog

    仓库xknower.github.io负责显示网站内容，不需要改动；日常开发和新增内容，都在仓库xknower-blog中，并通过 deploy. sh 命令，将代码发布到仓库 xknower.github.io

    1. vi deploy. sh
    #!/usr/bin/env sh

# 确保脚本抛出遇到的错误
set -e

# 生成静态文件
yarn build

# 进入生成的文件夹
cd dist

# 如果是发布到自定义域名
echo 'jiyuan.live' > CNAME

git init
git add -A
git commit -m 'deploy'

# 如果发布到 https://xknower.github.io/<REPO>  REPO=github上的项目
git push -f https://github.com/xknower/xknower.github.io.git master

cd -

    1. 访问网站 https://xknower.github.io/

    2. 购买域名并配置, 解析域名 www.jiyuan.live 和 @.jiyuan.live (*.jiyuan.live) 到 185.199.109.153 (ping xknower.github.io)

    3. 仓库 xknower.github.io , Settings > Custom domain 把域名 jiyuan.live 添加进去

    4. 访问网站 http://www.jiyuan.live/
